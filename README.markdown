Tapestry.tmbundle
======

This TextMate bundle is in development. Currently it has the following features:

 * tml syntax highlighting 
 * Tag transformation from T4 to T5 tml (`<span jwcid="@If"…>` => `<t:if …>`)
 * Tab snippets for:
   * if
   * loop
   
## Requirements

  * Nokigiri bundle 
  
      <pre>$ sudo gem install nokigiri</pre>
